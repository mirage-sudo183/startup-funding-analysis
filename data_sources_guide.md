# Complete Data Sources Guide for Startup Funding Research

## Where to Find ALL Funding Rounds

The comprehensive CSV I've created contains **200+ major funding rounds** from 2025. However, to get truly comprehensive data on ALL funding rounds (there were **10,000+ deals globally** in 2025), you'll need access to professional databases.

---

## Tier 1: Premium Databases (Comprehensive Data)

### 1. Crunchbase Pro/Enterprise
- **URL**: https://www.crunchbase.com
- **Coverage**: 10,000+ funding rounds per year
- **Data**: Company, amount, stage, investors, valuation, date
- **Cost**: $49/month (Pro) to Enterprise pricing
- **Best For**: Most comprehensive startup funding data

### 2. PitchBook
- **URL**: https://pitchbook.com
- **Coverage**: Complete VC/PE deal data globally
- **Data**: Deal terms, valuations, investor details, exits
- **Cost**: Enterprise (typically $20K+/year)
- **Best For**: Professional investors, detailed deal terms

### 3. CB Insights
- **URL**: https://www.cbinsights.com
- **Coverage**: Global startup funding with analytics
- **Data**: Deals, trends, market maps, investor tracking
- **Cost**: Enterprise pricing
- **Best For**: Market intelligence and trend analysis

### 4. Dealroom
- **URL**: https://dealroom.co
- **Coverage**: Strong European focus + global data
- **Data**: Funding rounds, valuations, exits
- **Cost**: Subscription-based
- **Best For**: European startup ecosystem

### 3. TopStartups.io
- **URL**: https://topstartups.io
- **Coverage**: VC-backed startups by industry

- **Best For**: Researching VC-backed companies

### 4. Tracxn

- **Coverage**: Global with strong India/Asia coverage

## How to Get Complete Data

### Option 1: Crunchbase Export
1. Sign up for Crunchbase Pro ($49/mo)
2. Use Advanced Search to filter by:
   - Funding Date: Jan 1, 2025 - Dec 31, 2025
   - Funding Type: Seed, Series A, B, C, etc.
   - Location: As needed
3. Export to CSV (up to 10,000 rows)

### Option 2: Combine Free Sources
1. Download from GrowthList (weekly updates)
2. Cross-reference with Fundraise Insider
3. Add Crunchbase News weekly roundups
4. Supplement with regional sources (Inc42, EU-Startups)

### Option 3: API Access
- Crunchbase API: For programmatic access
- PitchBook API: Enterprise customers
- CB Insights API: Enterprise customers
